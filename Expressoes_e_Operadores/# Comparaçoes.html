# Comparaçoes Servem para comparar dois valores e/ou variáveis. - Igual - `==` - Diferente - `!=` - Menor - `<` - Menor igual - `<=` - Maior - `>` - Maior igual - `>=` Uma comparação sempre retornará verdadeiro ou falso: ```javascript 1 > 2 1 == 1 "A" == "A" "A" < "B" ``` Para variáveis: ```javascript const idade = 16 const limite = 18 idade >= limite ``` # Comparação estrita (idêntico) O JavaScript tenta converter dados, por isso o uso do `==` pode ocasionar problemas. Ex: ```javascript "1" == 1 1 == true null == undefined ``` Para solucionar este problema utilizamos a comparação estrita (idêntico). ```javascript "1" === 1 1 === true null === undefined null !== undefined ```
